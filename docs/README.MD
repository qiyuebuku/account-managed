
# 账号管理器

使用python+pyqt5开发，帮你自动管理你的账号：邮箱、qq、微信

## 目前完成了邮箱管理
1. 邮件群发
2. 支持普通文本和html格式的内容
4. 支持添加附件
5. 完成自动回复功能
6. 支持对所有人回复或者对指定的邮箱回复
